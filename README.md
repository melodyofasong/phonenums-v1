# Extract Phone Numbers from Text

## Features
- Extracts phone numbers from large text files.
- Exports unique, accurate **Indian** numbers to CSV.
- Input: Provide .txt file path via CLI.
- Output: CSV appears in current directory.

**Note:** Works exclusively for Indian formats (+91 or 10-digit).

this is just a learning project btw :D

## Requirements
- Python 3+
- `pip install phonenumbers`

## Usage
```bash
python extract_phones.py
